# Skill: 中文文档与沟通规范

## 适用范围
- 产品规划讨论 (Product Planning)
- 更新 PRD (Product Requirements Document)
- 编写 PLAN (开发计划)
- 其他项目管理类文档

## 规范细则
1.  **主要语言**: **强制使用中文**。
2.  **英文例外**:
    - 专有名词 (如: React, FastAPI, QFQ, ETF)
    - 常见的行业缩写 (如: MVP, PRD, API)
    - 代码片段、变量名、文件路径
3.  **执行要求**: 在涉及上述场景时，无需用户额外提醒，默认输出中文。
