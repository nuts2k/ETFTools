# Claude Code 工作指引

⚠️ **【强制】开始任何工作前，必须先阅读 `AGENTS.md`**

## 核心文档位置

**项目概况、开发规范、技术栈、代码导航等所有核心信息均在 `AGENTS.md` 中维护。**

请在执行任何开发任务前，务必：

1. **阅读 `AGENTS.md`** - 了解项目身份、技术栈、强制性开发规范
2. **遵守第 4 节强制性开发规范** - 特别注意：
   - 数据一致性规范（前复权 QFQ）
   - UI/UX 强制规范（红涨绿跌、移动端优先）
   - 性能强制规范（防抖、缓存策略）
   - 安全规范（输入验证、SQL 注入防护、XSS 防护）
   - 测试规范（单元测试、API 测试）
   - **文档维护规范（代码变更必须同步更新文档）**
3. **参考第 5-9 节** - 快速定位核心代码路径、API 接口、配置文件

## 文档维护要求

根据 `AGENTS.md` 第 4.6 节规定：

- ✅ 代码变更与文档更新必须在同一个 commit 中提交
- ✅ 根据变更类型检查并更新对应文档（参见 AGENTS.md 第 4.6 节的检查清单）
- ❌ 禁止"先提交代码，后补文档"
