# Changelog

## [2026-02-04] - 告警配置默认值更新

### 变更内容

- **告警配置 API 默认值调整**
  - `max_alerts_per_day` 默认值从 20 提升到 100
  - 用户仍可在 1-100 范围内自定义配置
  - 现有用户配置不受影响（保持用户设置的值）

### 技术细节

- 修复了 API 端点 `AlertConfigRequest` 与数据模型 `UserAlertPreferences` 之间的默认值不一致性
- 相关文件：
  - `backend/app/api/v1/endpoints/alerts.py:29` - API 请求模型
  - `backend/app/models/alert_config.py:81` - 数据模型

### 影响范围

- ✅ 完全向后兼容
- ✅ 无需数据库迁移
- ✅ 无需 Docker 配置变更
- ✅ 仅影响新创建的告警配置的默认值
